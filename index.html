How the Web Works

HTTP (Hypertext Transfer Protocol)

HTTP is the set of rules for transferring files (text, graphic images, sound, video, and other multimedia files) on the World Wide Web. As soon as a Web user opens their Web browser, the user is indirectly making use of HTTP.

HTML Tag

An HTML Tag is a code that describes how a Web page is formatted.
  • em content-italic,Renders as emphasized text
  • a defines a hyperlink
  • br inserts a single line break
  • p tag defines a paragraph.
  • span tag is used to group inline-elements in a document
  • div tag defines a division or a section in an HTML document.
  • img tag defines an image in an HTML page.

HTML Attribute

An HTML Attribute is a modifier of an HTML element
  • src
  • width
  • height
  • title
  • style
  • value

HTML Element

An HTML element is an individual component of an HTML document or web page

Inline vs. Block

Inline elements render inline with the HTML block elements create an invisible block around the content.

Whitespace

Whitespace is the blank space in an HTML page. Relevant tags are br, p, span,div

CSS Basics

What is CSS ?

CSS is Cascading Style Sheets.............
  • CSS defines how HTML elements will be displayed.
  • External style sheets are stored in CSS files(.css)
  • CSS helps programmers avoid repetition.

CSS Syntax

CSS Syntax rule sets consists of a selector and a declaration block.
  • ex.1- h1 {color:blue; font-size:12px;}
  • ex.2- p {color:red;text-align:center;}

CSS Selectors

CSS Selectors are...........
  • Element selector-selects elements based on the element name (p {})
  • ID selector- uses the id attribute of an HTML element to select a specific element (#para {})
  • Class selector-selects elements with a specific class attribute (.center {})

Types of Stylesheets

  • Internal CSS stylesheet-An internal style sheet may be used if one single page has a unique style.Internal styles are defined within the style element, inside the head section of an HTML page
  • External CSS stylesheet-you can change the look of an entire website by changing just one file. Each page must include a reference to the external style sheet file inside the link element. The link element goes inside the head section.
  • Inline Stylesheet-An inline style may be used to apply a unique style for a single element.An inline style loses many of the advantages of a style sheet (by mixing content with presentation). Use this method sparingly!To use inline styles, add the style attribute to the relevant tag. The style attribute can contain any CSS property. The example shows how to change the color and the left margin of a h1 element. ex. h1 style="color:blue;margin-left:30px;">This is a heading. /h1
